# ATM System in Python

A simple console-based ATM system implemented using Python and Object-Oriented Programming concepts.

## Features
- PIN generation
- Deposit money
- Withdraw money
- Balance check
- Menu-driven program

## Concepts Used
- Classes and Objects
- Encapsulation (private variables)
- Getter and Setter methods
- Conditional statements

## How to Run
1. Clone the repository
2. Run the Python file:

```bash
python atm.py
